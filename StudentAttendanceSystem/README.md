# Student Attendance System

Simple Java project skeleton for tracking students and attendance with CSV-backed storage.

Folders:
- `src/` - Java source files by package
- `data/` - CSV data files
- `reports/` - generated textual reports

Run:
1. Compile the Java sources (javac) with the appropriate classpath.
2. Run `ui.MainMenu` (java ui.MainMenu) from the project root.
